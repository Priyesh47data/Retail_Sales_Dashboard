# Retail Sales Performance Dashboard (Power BI)

## Project Overview
This project is an end-to-end Retail Sales Performance Dashboard built using Power BI.  
It analyzes business performance over a 2-year period and provides insights into sales, customers, and products.

---

## Objectives
- Analyze overall sales performance  
- Track customer growth  
- Identify top-performing products  
- Monitor monthly and yearly trends  
- Build an interactive dashboard  

---

## Dataset Description
The project uses 4 datasets:

### 1. Sales Data
- OrderID
- OrderDate
- CustomerID
- ProductID
- Quantity
- Discount
  

### 2. Customer Data
- CustomerID
- CustomerName
- City
- Segment
 

### 3. Product Data
- ProductID
- ProductName
- Category
- UnitPrice

### 4. Custome Calendar
- Date
- Month
- Quarter
- Year

---

## Data Modeling
- Relationships created using:
  - CustomerID  
  - ProductID
  - OrderID
  - OrderDate 
- DAX used for calculations  

---

## Key KPIs
- Total Sales  
- Total Customers  
- Quantity Sold  
- Average Order Value  

---

## Dashboard Features
- Monthly Sales Trends  
- Top Products & Categories  
- Country-wise Sales  
- Customer Insights
